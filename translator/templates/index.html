📥 Cách tích hợp model WLASL (dxli94/WLASL)

  1. Tải file trọng số đã huấn luyện từ Google Drive:
    WLASL pre-trained weights (.pth.tar)
  2. Giải nén — bạn sẽ thấy thư mục I3D/archived/ chứa các file nslt_100.pth.tar, v.v.
  3. Tạo thư mục models/wlasl/ trong project và sao chép file .pth.tar vào đó:
    mkdir -p models/wlasl && cp nslt_100.pth.tar models/wlasl/
  4. Khởi động lại server: python manage.py runserver

Nguồn: github.com/dxli94/WLASL  ·  Tham khảo models/wlasl_class_list.txt để xem danh sách 100 ký hiệu được nhận diện.
